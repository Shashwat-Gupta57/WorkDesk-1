# WorkDesk UI Context

Product Name:
WorkDesk

Organization:
GoBitsNBytes Prayagraj

Purpose:

WorkDesk is a knowledge archive, collaboration workspace, and organizational operating system.

The product serves as the permanent institutional memory of the organization.

Primary users are:

* Marketing Leads
* Creative Leads
* Developers
* Team Members
* Administrators

The interface should feel like a blend of:

* Linear
* Notion
* Obsidian
* GitHub
* Google Drive

Avoid copying any product directly.

---

# Design Philosophy

Professional

Focused

Dense but not cluttered

Fast

Information-first

Minimal visual noise

Built for daily usage

No playful startup aesthetics.

No excessive gradients.

No glassmorphism.

No neumorphism.

The UI should feel like a serious productivity application.
